# Setup

### initialize app
```
$ npx create-expo-app <NAME>
```

### dependencies
```
$ npx expo install react-dom react-native-web @expo/webpack-config
$ npx expo install @expo-google-fonts/outfit expo-font
```

### start app
```
$ npx expo start
```

### start app ( for WSL2 )
```
$ npx expo start --tunnel
```
